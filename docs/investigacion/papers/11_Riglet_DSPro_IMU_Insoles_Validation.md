## Abstract

**Background:** Wireless wearable insoles are interesting tools to collect gait parameters during daily life activities. However, validation must be performed specifically for each type of insole on a big dataset in ecological situations.

**Objetivo:** Evaluar la **validez de criterio y la fiabilidad test-retest** de los parámetros de marcha obtenidos con plantillas inteligentes vs sistema de motion capture.

**Métodos:**
- **30 participantes sanos**
- Dispositivo: **DSPro® insoles** (con IMU embebido) — fabricante: Zhortech SAS
- Comparado contra motion capture system (Conventional Gait Model v2.5)
- Dos condiciones: pista de 10 metros y treadmill (EF 1800 Medical Développement)
- Tres velocidades distintas (lenta, cómoda autoseleccionada, rápida)
- Calibración del sujeto: 3 segundos de pie quieto antes de cada condición
- Métrica estadística: Intraclass Correlation Coefficient (ICC)

**Resultados:**
- **Fiabilidad >0.70 para todas las variables (p < 0.001)** excepto minimum toe clearance (ICC < 0.50) con motion capture en overground
- Independientemente de velocidad/condición, **validados con ICC > 0.90 (p < 0.001):**
  - Speed
  - Cadence
  - Stride Length
  - Stride Time
  - Stance Time
- Parámetros con correlación > 0.75 (buena validez):
  - Velocidad, cadencia, longitud y tiempo de zancada
  - Tiempo de apoyo (stance), tiempo de oscilación (swing)
  - Ancho de zancada, tiempo de doble apoyo
  - Foot progression angle y plantar flexion foot in
- **No validados:** loading time en treadmill a velocidad lenta (ICC < 0.70)

**Conclusión:** Este estudio destaca **buena validez de criterio y fiabilidad test-retest** de los parámetros espacio-temporales medidos con plantillas wearable, abriendo nuevas posibilidades para mejorar el manejo clínico de pacientes usando análisis de marcha en actividades diarias.

**Keywords:** gait analysis, insole, validation, test-retest reliability, criterion validity, 3D motion analysis

## Relevancia para tesis
**Referencia clave de validación de plantillas con IMU embebido**:
- Es exactamente el tipo de dispositivo objetivo (plantilla inteligente con IMU)
- Metodología de validación rigurosa replicable
- Aporta umbrales de calidad esperables (ICC > 0.90 para parámetros básicos)
- Identifica qué parámetros son fiables y cuáles no (e.g., toe clearance es complicado)
- Compara overground vs treadmill — útil para definir protocolo de evaluación
- DSPro® como dispositivo comercial de referencia
